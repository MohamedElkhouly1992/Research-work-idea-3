# Start here

## Fastest route

1. Install Python 3.11 or 3.12.
2. On Windows, double-click `run_local.bat`.
3. On Linux/macOS, run `./run_local.sh`.
4. Open the **Run** tab and press **Run HVAC–BMS simulation**.
5. Review **Dashboard**, **Time series**, **Alarms**, **Benchmark**, and **Validation**.
6. Export Excel, PDF, figures, and CSV files from **Downloads**.

The demonstration building is already configured with six zones, three AHUs, a central chilled-water plant, S0-S3 BMS strategies, degradation, FDD, and condition-based maintenance.

For the actual building, first replace the demonstration zone/AHU/plant parameters and upload measured or EPW-derived weather data. Use the generated JSON configuration as the reproducible scenario definition for each manuscript run.
